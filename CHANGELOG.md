## [1.0.0] - 2026-02-19

### Added

- Initial release of Agile PM Mode
- Custom command `/agile-plan [project-description]` for project planning
- 6-artifact generation pipeline:
  - USER_PERSONAS.md
  - USER_JOURNEYS.md
  - EPICS.md
  - USER_STORIES.md
  - ACCEPTANCE_CRITERIA.md
  - SPRINTS.md
- Evidence-based analysis of codebase
- Repo-aware artifact generation
- Execution-optimized output focus

### Features

- Read access to analyze codebase
- Write access restricted to Markdown files only
- Command execution support for CLI operations
